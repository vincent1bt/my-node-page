Alzheimer's disease Image detection

Alzheimer's is a progressive disease that destroys memory, as the disease advances patients loss cognitive abilities in such a way that they can not live a normal life anymore.

The disease has no cure but medication may improve symptoms.

After presenting signs of Alzheimer's patients can take a positron emission tomography (PET) in order to make a diagnostic, in this kind of tomography the patient has to consume a radiopharmaceutical called fludeoxyglucose F-18, this radiopharmaceutical helps get more information about the patient's brain condition.

The images used to train this model are from the ADNI dataset (Alzheimerâ€™s Disease Neuroimaging Initiative) adni.loni.usc.edu. I converted the 3d images from the dataset to 2d jpg images.

This model outputs the class activation maps as well. Due to these maps, we can see the most important parts of the brain that helped the model to classify the image as cognitive normal or dementia (Alzheimer's disease).

This neuronal network uses TensorflowJS. Therefore everything works in the browser and the images are not send to a server. The model has a sensitivity of 99% and a specificity of 99%. The code of this project is available as a Jupyter notebook.
